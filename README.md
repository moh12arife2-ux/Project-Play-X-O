📋 Overview
A classic Tic-Tac-Toe game developed using C# and Windows Forms (WinForms). The application features a clean, user-friendly graphical interface (GUI) where two players can enter their names and compete against each other.

🚀 Features
Player Registration: Custom name inputs for both Player 1 and Player 2.

Dynamic UI: Interactive buttons that respond to player moves.
